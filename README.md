## Installation

### Download

```
$ git clone https://github.com/Tatzyr/tut_lecture_bot.git
```

### Initial Setup

```
$ cd tut_lecture_bot
$ cp settings.yml.sample settings.yml
$ vi settings.yml
$ bundle install
$ bundle exec ruby migration.rb
```

### Run

```
$ bundle exec ruby tut_lecture_bot.rb
```
